## Install

#### Requirements

- Install Git
- Install PHP 8.1
- Install Composer
- Install Node js >= 16.x

### Get Started
- First, create an environment file with this command `cp .env.example .env`
- After that, follow this command:
####
    composer install
    php artisan key:generate
    npm install
    npm run build

- Then, you can start the project that hosted at `http://localhost:8000` with command:
####
    php artisan serve
